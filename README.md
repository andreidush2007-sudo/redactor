# redactor
Основной файл CircuitElement.sln - открывает проект.
CitrcuitElement - каталог с полной реализацией самого редактора: Components содержит описание всех элементов электрической схемы на уровне кода, Resoursec хранит файлы в формате pnj для элементов, CircuitEditorForm.cs отвечает за графичекую реализацию на форме, CircuitElement.cs отвечает за описание всех действий на форме, Program.cs - main файл для через который проходит запуск, ResourceHelper - помогает подтянуть фотографии из Resuorces для Components.
CircuitEditor.Test - каталог с unit тестами. Проходит проверка создания элементов, их соединения, удаления, поворота, а также сохранение и загрузка схем.
